# Voice-Enabled-Battery-Status-Monitor
This Python-based system utility uses text-to-speech technology to audibly announce the current battery status of the system. It retrieves real-time battery information such as charge percentage and charging state. This project demonstrates practical usage of Python libraries for system monitoring and accessibility-focused automation.
